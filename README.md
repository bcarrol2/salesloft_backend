# salesloft

Steps to run this app

1. cd into the salesloft_app directory
2. `bundle install`
3. `yarn` 
4. setup the database with rails db:create, rails db:migrate, & rails db:seed
5. `rails s` and navigate to http://localhost:3000/users in your browser
